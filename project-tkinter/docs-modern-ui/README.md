# Wariant A: Tkinter Modern UI

To jest oddzielny wariant projektu z zachowaniem desktopowego stosu Python + Tkinter, ale z nowym podejściem do UI/UX.

## Cel
- Zostawić backend i logikę aplikacji w Pythonie.
- Uporządkować layout i styl `ttk`.
- Dodać nowoczesny wygląd (spacing, typografia, hierarchia sekcji, spójne komponenty).

## Start
1. Pracujesz w `variants/tkinter-modern-ui`.
2. Uruchamiasz punkt wejścia:

```powershell
python app_entry.py
```

## Struktura
- `app_entry.py` - prosty launcher wariantu.
- `docs/ui-modernization-plan.md` - plan etapów modernizacji.

## Uwaga
Ten wariant **nie modyfikuje** produkcyjnych plików w katalogu głównym.
