# Plan modernizacji Tkinter UI

## Etap 1: Design tokens
- Zdefiniować jedną paletę kolorów i skale spacingu.
- Zastąpić rozproszone style jednym modułem stylu.

## Etap 2: Layout
- Ujednolicić grid i marginesy kart.
- Ograniczyć gęstość formularzy (więcej odstępów, logiczne grupy).

## Etap 3: Komponenty
- Spójne style przycisków: primary/secondary/danger.
- Spójne pola wejściowe i etykiety pomocnicze.

## Etap 4: UX
- Lepsze komunikaty statusów i błędów.
- Mniej modalnych popupów, więcej informacji inline.

## Etap 5: Dostępność
- Skróty klawiaturowe.
- Kontrasty i czytelność fontów.
